Glossary

Elements
An element is a basic unit of HTML that consists of a start tag, content, and an end tag. Elements are used to structure and format the content of a webpage.
Tag
A tag is a markup element that is used to define the start and end of an element. Tags are surrounded by angle brackets (< and >) and are used to enclose the element's content.
Attribute
An attribute is a property of an element that provides additional information about the element. Attributes are specified in the start tag of an element and are usually composed of a name and a value.
